# SwiftyVIN 1.0

A Swift version of VIN validations using Swift 4.2

## Getting Started

```
import SwiftyVIN
```

```
let swiftyVin: SwiftyVIN = SwiftyVIN("")
```

Then

```
swiftyVin.isValid()
```
